## 一、大模型动态

### Anthropic · Claude 4 技术报告发布

Anthropic 今日公开 Claude 4 系列的训练技术细节，这是其首次系统性披露模型架构改进。

**核心变化：**

- **对齐框架升级**：第三代 Constitutional AI（CAI）引入多智能体辩论机制，模型在生成答案时会主动构建反驳观点并自我校验，有害内容拒绝率提升 38%
- **上下文扩展至 200K**：通过稀疏注意力机制实现，推理时内存占用仅增加 12%，长文档理解任务得分较上代提升 22 个百分点
- **代码能力跃升**：SWE-bench 得分达 67.3%（Claude 3.5 Sonnet 为 49.0%），在真实 GitHub Issue 修复任务中表现尤为突出

Claude 4 Opus 目前仅对 API 用户开放，预计 Q3 进入 Claude.ai 消费端。Sonnet 和 Haiku 版本暂未披露时间表。

### OpenAI · GPT-4o 实时能力重大更新

GPT-4o 推送 `2026-05-28` 版本，重点优化了实时多模态交互体验。

**新增能力：**

- **实时视觉理解**：通话中可持续分析摄像头画面，支持 "帮我看一下这段代码有没有问题" 类指令，识别延迟约 180ms
- **代码沙箱集成**：对话中触发的 Python 代码可在隔离环境中直接执行，结果实时返回，免去用户切换 Interpreter 的步骤
- **语音延迟降至 230ms**：较上一版本（320ms）下降 28%，接近正常人类对话反应阈值（200ms 以内）

Realtime API 同步支持上述功能，定价不变。开发者文档已更新。

### Google DeepMind · AlphaFold 3 蛋白质设计扩展

DeepMind 发布 AlphaFold 3 更新，将原有蛋白质结构预测能力扩展到**从头设计**场景：

- 支持输入功能描述（"能结合 ATP 的催化位点"），模型直接生成满足条件的蛋白质序列
- 新增核酸-蛋白质共同设计模块，对 CRISPR 基因编辑工具的优化实验中准确率达 84%
- 与 10 家制药公司签订合作协议，用于早期药物靶点筛选

研究者可通过 AlphaFold Server 免费使用基础功能，商业 API 另行定价。

## 二、开源社区

### Meta · Llama 3.3 70B 正式发布

**下载地址**：Hugging Face `meta-llama/Llama-3.3-70B-Instruct`

Meta 今日将 Llama 3.3 70B 正式推入开源（Apache 2.0 许可），与 Llama 3.1 405B 参数量相差 5 倍，但关键指标持平甚至反超：

| 基准测试 | Llama 3.3 70B | Llama 3.1 405B | GPT-4o mini |
|---------|--------------|----------------|-------------|
| MMLU | 86.0 | 87.3 | 82.0 |
| HumanEval | 88.4 | 89.0 | 87.2 |
| MATH | 77.0 | 73.8 | 70.2 |

**部署参考**：INT4 量化后约占 40GB 显存，RTX 4090 单卡可运行，推理速度约 42 token/s。vLLM 和 llama.cpp 均已完成适配。

### Mistral · Magistral 工具调用改进

Mistral 发布 Magistral Medium 补丁版本，专项修复函数调用（function calling）中的格式幻觉问题。在内部测试集中，结构化输出格式遵从率从 91.2% 提升至 97.8%，对 RAG 管道和 Agent 工作流影响明显。

## 四、研究前沿

**《Scaling Laws for Reward Model Overoptimization》**（DeepMind）

研究发现 RLHF 中奖励模型过拟合（reward hacking）在参数量超过 70B 后呈现非线性加剧，提出"奖励模型集成对齐"方法，在 Gemini 1.5 Pro 上验证可将对齐稳定性提升 31%。

**《Chain-of-Draft: Thinking Faster by Thinking Less》**（Zoom AI Research）

提出 Chain-of-Draft（CoD）方法：要求模型用极简草稿（每步不超过 5 词）替代完整 CoT，在 GSM8K 和 date understanding 任务上准确率与 CoT 持平，token 消耗减少 76%，对实时推理场景有较大实用价值。
